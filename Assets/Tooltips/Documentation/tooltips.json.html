Файл tooltips.json

Файл с данными для выносок, содержится в папке StreamingAssets.

Данные представляют собой один json-объект для каждой выноски. json-объекты соотносятся с выносками через имя: поле name в компоненте Tooltip должно совпадать с полем name в json-объекте.

В начале работы программы данные считываются из этого файла и передаются выноскам, заменяя данные, которые уже есть в выноске. Выноски, данных к которым нет в файле, сохраняют свои исходные данные.

Структура файла

        {
            "tooltips": [
                {
                    "name": "Имя выноски",
                    "data": [
                        {
                            "type": "title",
                            "stringData": "Заголовок выноски"
                        },
                        {
                            "type": "text",
                            "stringData": "Тект выноски"
                        },
                        {
                            "type": "progress",
                            "stringData": "Описание полосы прогресса",
                            "numberData": 0.5
                        }
                    ]
                }
            ]
        }
