class TooltipMouseTrigger

Контроллер для выноски. Показывает выноску при наведении курсора на объект в мире.

Компонент должен находиться на одном объекте с коллайдером. Показывает выноску только если скрипт активен (isActiveAndEnabled = true).

Свойства

tooltip

Выноска, которой управляет этот компонент

appearDelay

Задержка перед появлением выноски
