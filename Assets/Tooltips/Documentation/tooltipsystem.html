class TooltipSystem

Система для информационных выносок.

Содержит список выносок, ссылку на камеру и ссылки на префабы. Загружает данные для выносок из json файла, затем передает эти данные выноскам. Каждая выноска использует эту систему для доступа к камере (для расчета положения на экране) и других целей.

Для использования нужно заполнить поля компонента в редакторе Unity.
